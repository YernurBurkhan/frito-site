# FRITO ✌ - пародийный микросайт ордена

Одностраничный лендинг в стиле тёмный неон-глитч. Один файл `index.html`
со встроенными CSS и JS. Без сборки, без фреймворков, без зависимостей
(кроме одного веб-шрифта с фолбэком на системный).

## Структура

```
Сайт/
├── index.html              # весь сайт (HTML + CSS + JS)
├── README.md
└── assets/
    ├── frito-logo.png      # неон-глитч лого (главный визуал блока 1)
    └── frito-hand.mp4      # зацикленная неоновая рука (блок 2)
```

## Как запустить

Просто открой `index.html` двойным кликом в браузере. Видео и PNG лежат
рядом в `assets/`, поэтому всё подхватится без сервера.

Если хочется локальный сервер (например, для корректной аналитики GTM),
подними любой статик-сервер из папки `Сайт` и открой `http://localhost`.

## Куда вставить GTM-ID

В `index.html` стоит placeholder `GTM-XXXXXXX` в **двух местах** -
замени оба на свой ID контейнера:

1. В `<head>`, внутри сниппета Google Tag Manager.
2. В `<body>`, в `<noscript>` (первая строка после `<body>`).

После замены проверь в GTM Preview, что прилетают события:

| Событие            | Когда                          | Поля                                            |
|--------------------|--------------------------------|-------------------------------------------------|
| `frito_page_view`  | при загрузке страницы          | -                                               |
| `frito_click`      | клик по кнопке (до перехода)   | `button_id` (A / B), `button_text`, `link_url`  |

`gtag` напрямую не подключён - GA4 настраивается внутри GTM, как и просили.
`window.dataLayer` объявляется до загрузки GTM.

## Что внутри (по блокам)

**Блок 1 (hero):** лого FRITO крупно, с параллаксом по курсору и screen-блендингом
(чёрный постер сливается с фоном, водяной знак обрезан через `clip-path`).
Слоган-eyebrow «мкмкмк с 2020 · Айуан-сертифицировано», кнопка А «мкмкмк →»
на Instagram reel.

**Блок 2 (ритуал):** видео неоновой руки (`autoplay muted loop playsinline`,
без контролов) в неоновом кольце, глитч-заголовок «Освой священное движение»,
абсурдное описание ритуала, кнопка Б «Неменемді үрлетіп алайын 🔥» на YouTube.

## Пасхалки

- Клик по лого FRITO меняет слоган на случайный мем + микро-глитч.
- Наведение на кнопку: вздрог + всплывает «Айуан».
- Уход со вкладки меняет title на «Спать или не спать, вот в чём вопрос».
- Тот же девиз спрятан мелким текстом в футере.
- По фону вяло дрейфуют рандомные мем-слова.

## Доступность и адаптив

- Полностью адаптивно: desktop - две асимметричные секции, мобайл - одна колонка.
- `clamp()` для шрифтов, `min-h-100dvh` (без прыжков адресной строки на iOS).
- `prefers-reduced-motion`: все тяжёлые анимации (глитч, дрейф, параллакс,
  дыхание кольца) отключаются, остаётся чистая статика.
